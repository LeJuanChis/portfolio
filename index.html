Mis conocimientos

Durante mi tiempo como programador he experimentado con las siguientes tecnologias:

html

Como desarrollador web tengo muy en claro que html es una parte fundamental para el maquetado y el diseño de una pagina. Absolutamente todos mis proyectos web tienen algo de html en su estructura.

php

Php es uno de los lenguajes mas usados en el mundo y es con el que yo me empece en este maravilloso mundo. Iniciando desde las bases hasta la programacion orientada a objetos y MVC

css

El lenguaje por medio del cual le damos estilos a nuestras paginas web. Aunque no soy un fanatico del front-end, he ido escalando hasta llegar a temas como grid, flexbox y diseño responsive.

java

Realice mi proyecto de la tecnologia en este lenguaje, sin embargo es el que menos experiencia tengo y sinceramente el que menos me gusta. Aunque gracias a su estricta sintaxis me ayudo a comprender y reforzar temas como POO y MVC.

javascript

Este lenguaje me gusta bastante y aunque no soy un experto en el, si tengo conocimientos basicos en DOM y ES6. Personalmente me gusta mucho y pienso que tiene muy merecido el puesto del lenguaje mas usado.

mysql - workbench

El sistema gestor de bases de datos por excelencia. Lo he usado para mis bases de datos desde hace mucho y me gusta bastante. Con el he aprendido cosas como normalizacion, tipos de datos y relaciones.

sass

Me gusta definirlo como: "Css con superpoderes". Es un preprocesador de Css con el que podemos crear hojas de estilos con caracteristicas que no tiene css navito. Debe ser compilado y para esto yo uso Guls.js.

sql

El complemento de mysql, en pocas palabras es el lenguaje que nos ayuda a manipular datos de nuestra base de datos con el fin de mostrarselos al cliente o hacer algun tipo de logica con el lenguaje de programacion
